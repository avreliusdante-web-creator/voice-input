Voice input

Browser extension: voice → text in social media and messengers

О проекте

Voice input — Chrome and Edge extension that converts speech to text и позволяет быстро вставлять его в поля ввода на сайтах (ВКонтакте, Telegram Web, WhatsApp Web, Twitter/X, Facebook, Instagram, Gmail, Outlook и др.). Не нужно переключаться в другие приложения: говорите прямо на странице — текст появляется в панели и одной кнопкой вставляется в открытый диалог.

Расширение не отправляет голос на сторонние серверы разработчика: распознавание речи выполняется через встроенный в браузер Web Speech API (в Chrome/Edge — сервисы Google). Локально хранятся только файлы расширения; персональные данные не собираются.

Принцип работы

Расширение работает только на указанных в манифесте сайтах (VK, Telegram Web, WhatsApp Web, Twitter/X, Facebook, Instagram, LinkedIn, Gmail, Outlook и др.).

Content script (кнопка и панель на странице)
На каждом из поддерживаемых сайтов внедряется скрипт, который:
  1. Добавляет в правый нижний угол экрана плавающую кнопку с иконкой микрофона.
  2. По клику открывает панель: выбор языка, кнопка записи, поле с текстом, кнопки-иконки (Исправить, Копировать, Очистить, Вставить в диалог), переключатель светлой/тёмной темы.
  3. Перед записью запрашивает доступ к микрофону через getUserMedia({ audio: true }), затем запускает SpeechRecognition. Распознавание непрерывное (continuous), с промежуточными (interim) и финальными (final) результатами; финальный текст добавляется в «зафиксированную» строку, чтобы не дублировать слова.
  4. По кнопке «Вставить в диалог» текст вставляется в поле ввода на странице (приоритет — активный элемент или поле внизу экрана с подходящим placeholder/ролью).

Иконка расширения в панели браузера не открывает отдельное окно; весь интерфейс — только кнопка и панель на самих сайтах.

Технологии

Расширение построено на стандартных веб-технологиях и API браузера.

Manifest V3 Chrome Extension API Web Speech API (SpeechRecognition) getUserMedia (audio) HTML5 / CSS3 Vanilla JavaScript (ES5+ в content script)

Разрешение: activeTab. Дополнительные страницы или фоновый скрипт не используются.

Основные возможности

  • Распознавание речи в реальном времени (непрерывный режим, промежуточные и финальные результаты).
  • Много языков в панели: русский, украинский, английский, немецкий, французский, испанский, итальянский, португальский, китайский, японский, корейский и др.
  • Автоисправление пунктуации и заглавных букв после точки/вопросов/восклицаний.
  • Копирование в буфер обмена и вставка текста в поле сообщения на странице (с фокусом на целевое поле).
  • Поиск поля ввода: приоритет активному элементу, затем полям в нижней части экрана и с подходящими placeholder/role.
  • Стиль glassmorphism, адаптация под светлую/тёмную тему страницы, кнопка смены темы; иконки-кнопки с подсказками (title), анимация открытия/закрытия панели.

Экраны и сценарии

Плавающая кнопка — в правом нижнем углу страницы (только на поддерживаемых сайтах). Клик открывает панель.
Панель на сайте — выбор языка, кнопка записи, поле текста, иконки: Исправить, Копировать, Очистить, Вставить в диалог; в шапке — смена темы (светлая/тёмная) и закрытие.

Ограничения и требования

  • Работает в Chrome и Edge (Chromium); в Firefox Web Speech API по умолчанию отключён.
  • Требуется доступ к микрофону и (для распознавания в Chrome/Edge) интернет (используются сервисы браузера).
  • Расширение отображается только на сайтах, перечисленных в манифесте; на остальных страницах кнопка и панель не появляются.
